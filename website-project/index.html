Steps to Get Started

Step 1 - Theme Options

To begin customizing your site, go to Appearance -> Customizer and select Theme Options.

Step 2 - Setup Slider

To add a slider, go to Theme Options -> Homepage and choose page slider.

Step 3 - Create Homepage

To add featured content, go to Theme Options -> Homepage (Featured) and turn the switch on.
